# ClientFlow

Sistema full stack de gestão de clientes para freelancers e pequenos negócios.

## Status

Em desenvolvimento.

## Sobre o projeto

O ClientFlow é uma aplicação full stack para cadastro, organização e gerenciamento de clientes. O objetivo é praticar e demonstrar conhecimentos em desenvolvimento web completo, incluindo autenticação, API REST, banco de dados, interface responsiva e deploy.

## Tecnologias previstas

### Front-end

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- Context API

### Back-end

- Node.js
- Express.js
- MySQL
- JWT
- Bcrypt
- Arquitetura MVC

## Funcionalidades planejadas

- Cadastro e login de usuários
- Autenticação com JWT
- CRUD completo de clientes
- Proteção de rotas
- Cada usuário visualiza apenas seus próprios clientes
- Busca de clientes por nome
- Filtros por status
- Paginação
- Dashboard com métricas
- Foto do cliente
- Tema claro/escuro
- Deploy do front-end, back-end e banco de dados
